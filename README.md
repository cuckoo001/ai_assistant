# 🤖 AI-Assistant (轻量级智能对话助手)

一个基于 Python 开发的专业 AI 对话工具。它不仅能与你流畅聊天，还具备“下班自动总结”和“工业级异常重试”逻辑。

## ✨ 核心亮点

- **🧠 智能记忆管理**：内置滑动窗口算法，自动修剪历史对话，既保证了上下文连贯，又精准控制了 Token 消耗（省钱）。
- **📝 自动复盘总结**：当你输入 `退出` 时，AI 会自动提炼本次对话的精华，并保存至 `daily_notes.txt`。
- **🛡️ 安全隔离架构**：采用 `.env` 环境变量存储敏感信息，确保 API Key 不会随代码泄露。
- **⚙️ 健壮的重试机制**：面对网络波动或服务器繁忙，程序会自动尝试重连（上限 3 次），并实时反馈报错信息。
- **📅 详尽日志记录**：自动记录带时间戳的 `chat_log.txt`，方便随时翻阅原始对话。

## 🛠️ 环境要求

- **Python**: 3.8 或更高版本
- **依赖库**: `openai`, `python-dotenv`

## 🚀 快速上手    

**安装依赖**  
```bash
pip install openai python-dotenv
```
**配置 API Key**  
*在 ai_assistant 根目录下手动创建一个 .env 文件，内容格式如下：*
```Plantext
MOONSHOT_API_KEY=你的_Kimi_API_KEY
```

⚠️ 注意事项
请务必不要将含有真实 Key 的 .env 文件上传至 GitHub。

程序默认使用 kimi-k2.5 模型，如需更换请修改MODEL_NAME。
